Tujuan Website

Website ini dibuat sebagai bagian dari ujian mid semester mata kuliah Web Design. Tujuannya adalah untuk mempraktikkan penggunaan elemen-elemen HTML seperti tabel, form, gambar, dan link, serta penerapan class CSS untuk desain tampilan yang menarik.

Teknologi yang digunakan:

  • HTML (struktur halaman)
  • CSS (tata letak dan warna)
  • Tanpa JavaScript (karena fokus pada desain statis)
